# About this project

```
>npm install // instala dependencias
```

```
>npm run start // node
>npm run dev // nodemon
```

### Dependencias:

- Express

```
    npm install express
```

```
npm i morgan -D
```

- websocket

```
npm i socket.io
```

- Conexión con MongoDB

```
npm i mongoose
npm i connect-mongo
```

- Variables de entorno

```
npm install dotenv --save
```

- CORS

```
 npm i cors
```

- Pagination

```
 npm i mongoose-paginate-v2
```
